# Interview Invitation Form

A professional, frontend-only web application for generating customized interview invitations. This tool allows users to create formal, job-oriented invitation letters for interviews, suitable for recruiters, HR professionals, or hiring managers.

## Features

- **Interactive Form**: Input fields for company name, role, interview date/time, mode (In-Person/Virtual), contact email, and additional message.
- **Professional Design**: Clean, responsive UI with a business-appropriate color scheme and typography.
- **Dynamic Generation**: Generates a formal invitation letter based on user inputs.
- **Validation**: Ensures all required fields are filled before generating the invitation.
- **Mobile-Friendly**: Responsive design that works on desktop and mobile devices.
- **Frontend-Only**: No backend required; all functionality runs in the browser.

## Usage

1. Open `index.html` in a web browser.
2. Fill in the required fields in the form:
   - Company Name
   - Role
   - Interview Date
   - Interview Time
   - Interview Mode
   - Contact Email
   - Additional Message (optional)
3. Click "Generate Invitation" to create and display the professional invitation letter.
4. The generated invitation can be copied or printed for use.

## Technologies Used

- HTML5
- CSS3 (with responsive design)
- JavaScript (ES6+)

## Browser Compatibility

Works in all modern web browsers that support HTML5, CSS3, and JavaScript.

## Customization

The form and generated invitation can be easily customized by editing the HTML, CSS, and JavaScript in `index.html`.
